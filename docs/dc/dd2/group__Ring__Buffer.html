Ring buffer status enumerator.

Enumerator
NO_PTR 

Returned if null pointer was given as parameter.

NO_DATA 

Returned if there was no data to read in buffer.

NO_PLACE 

Returned if there was no place to write data in buffer.

Returned when write/read was succesfull/
